# LBTC Ecosystem Ontology

**Ontology** is the single source of truth for all LBTC Ecosystem projects.

It defines the fundamental concepts, relationships, and semantic rules that govern how data, actors, and systems interact across all implementations.

---

## What is This?

This repository contains the **universal model** for the LBTC Ecosystem:

- **ixbase** — Medical data formalization and archival
- **forks-life-node** — Cryptocurrency wallets and node deployment
- **legacycoin** — Blockchain, exchange, smart contracts
- **enterprise-management** — Business operations and automation

All four projects inherit from and extend this ontology.

## Quick Reference

### The Seven Pillars

| Concept | Role | Examples |
|---------|------|----------|
| **Block** | Immutable container of operations | Medical record, Transaction, Order |
| **Table** | Structured data collection | Patient registry, Account ledger, Token supply |
| **Operation** | Atomic state change | Diagnosis entry, Transfer, Trade execution |
| **Actor** | Entity that initiates operations | Human, Organization, Bot, Service, Node |
| **Node** | Validation and consensus point | Blockchain node, ixBase validator, Market node |
| **Semantics** | Meaning and interpretation rules | Medical terminology, Currency codes, Contract logic |
| **Consensus** | Agreement mechanism | Blockchain consensus, Data validation, Market settlement |
